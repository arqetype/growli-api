# Growli API

A Spring Boot REST API.

## Prerequisites

- Java 21+
- Maven 3.9+

## Getting Started

```
mvn spring-boot:run -Dspring-boot.run.profiles=dev
```
The API will be available at http://localhost:8080/

With this API you will also have a Swagger UI available at http://localhost:8080/swagger-ui/index.html only in development profile.

## Profiles
- dev
- prod
